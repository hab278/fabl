

Some of the path-builder helpers depend
on the following three environment variables
being set.

For specs to pass, they need to be set to these
exact values.

export FINDINGAIDS_FULL_DEFAULT_EXTENSION=html
export FINDINGAIDS_FULL_PATH=/findingaids/html
export FINDINGAIDS_FULL_HOST=dlib.nyu.edu


If you need to index to Solr but you're off campus, 
you'll need to tunnel, details in the solr.yml
